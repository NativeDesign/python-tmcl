Installation
============


Requirements
------------

- pyserial_



Install with pip
----------------

.. code-block:: bash

	$> pip install tmcl



Install without pip
-------------------

.. code-block:: sh

	$> git clone https://github.com/NativeDesign/python-tmcl.git
	$> cd python-tmcl
	$> python setup.py install




.. _pyserial: https://pythonhosted.org/pyserial/
